ABOUT

Hello! My name is John, a passionate and dedicated front-end web developer with a strong background in creating dynamic and user-friendly websites and applications. I specialize in the MERN stack (MongoDB, Express.js, React.js, Node.js) and have a keen interest in building efficient and scalable web applications.

I graduated with a Bachelor of Engineering (BEng) in Biomedical Engineering back in 2021. During my studies, which involved a great degree of programming and problem-solving, I discovered my passion for software development and knew I would segway into a career in web development upon graduation. Since then, I've completed an online web development bootcamp and built a couple projects in order to master the skills necessary to excel in this field.

The technologies I've currently been working with are:

  • JavaScript (ES6+), HTML, CSS
  • React.js, Node.js, Express.js
  • MongoDB
  • Git, Github

While I have some back-end knowledge, I plan on becoming versed with intermediate concepts such as performance optimization, CSS pre-processors, CSS-in-JS and TypeScript before delving into full-stack territory.

I am currently looking for a junior front-end developer position where I can contribute to exciting projects and continue to grow my skills. I am particularly interested in opportunities that allow me to work on innovative web applications and collaborate with experienced professionals in the field.

When I'm not coding, I enjoy exploring the latest web development trends, drinking tea, and playing sports or biking with friends.
